Realtime Chat App

Type: Personal Project

A full-stack real-time chat application that allows users to send messages to each other instantly. This project supports one-on-one chats, persistent message history, and a sleek WhatsApp-inspired interface. Users can login, view previous conversations, and initiate new chats by username. WebSockets are used for real-time communication.

Features:

  • Login and Register system with session storage
  • View chat history with multiple users
  • Start new chat by entering username
  • Realtime messaging using Socket.IO
  • Responsive design with clean UI similar to WhatsApp

Tech Stack:

  • Frontend: React, CSS Modules
  • Backend: Node.js, Express.js
  • Database: MongoDB (Mongoose)
  • Realtime: Socket.IO
