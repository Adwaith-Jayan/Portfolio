Code Arcade

Type: Personal Project

Code Arcade is an interactive web-based platform featuring a growing collection of mini-games designed to challenge and entertain users through logic, puzzles, and strategy. Built with modern web technologies, the arcade showcases seamless animations, clean UI, and engaging user experience. From classic problem-solving games to modern implementations like the 8 Puzzle and Tic Tac Toe with AI, Code Arcade emphasizes both fun and educational value. The project was created to demonstrate frontend engineering skills using React, Vite, and Tailwind CSS, and is continually expanding with new games and enhancements.

Features:

  • Collection of classic and custom mini-games
  • Built using React, Vite, and Tailwind CSS
  • Animated transitions between pages and games
  • Each game opens in a new interactive UI
  • Hover effects with live project previews
  • Project tiles with tool highlights and discover links
  • Under-development games show a custom video splash screen
  • Fully responsive design with keyboard and mouse support

Tools & Technologies:

  • Frontend: React.js, Tailwind CSS
  • Version Control: Git, GitHub
